CourseCatalog
======================

This repository is a collection of projects demonstrated at the RWDevCon. The instructor folder contains resources related to each stage of the demonstrations, lab and challenge exercise.

The demo project is a simple Course Catalog application that downloads and imports the entire contents of the Coursera course catalog. The demo project uses the following frameworks:

* Argo - JSON Parsing
* Swell - Logging
* Runes - Functional operators required for Argo
* Quick - BDD Testing in Swift
* Nimble - Expressive Matcher framework in Swift

To update the frameworks, use carthage.
